误区

  1. 滥用 float 布局
  2. 使用空标签闭合浮动
  3. 用 <br> 控制垂直间距
  4. 用 &nbsp; 控制字间距(两端对齐布局等)
  5. 直接在标签上用 style 属性写样式
  6. 随意甚至使用拼音给 Class，ID 命名
  7. 多 Class 症(滥用 OOCSS)
